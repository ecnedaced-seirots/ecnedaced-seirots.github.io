Go入門(13) - Switch文による条件分岐

February 6, 2022

Go言語のSwitch文による条件分岐例など。

 

基本構文。まぁ説明するまでもない。


switch 条件 {
case A:
    処理コードA
case B:
    処理コードB
case C:
    処理コードC
default:
    デフォルト処理
}

 

サンプル。引数を与えて、その値によって処理を分岐させる。

switch1.go

package main

import (
    "flag"
    "fmt"
)

func main() {
    flag.Parse()
    
    //条件の値となる引数を定義。
    city := flag.Arg(0)

    //引数の値に応じて処理
    switch city {
    case "Tokyo":
        fmt.Println("住んでいます。")
    case "Bangkok":
        fmt.Println("一度行ったことがあります。")
    case "NY":
        fmt.Println("行ったことないです。")
    default:
        fmt.Println("いつか行きたいです。")
    }
}

 

結果。

$ go run switch1.go Tokyo
住んでいます。

$ go run switch1.go NY
行ったことないです。

$ go run switch1.go Bangkok
一度行ったことがあります。

$ go run switch1.go Havana
いつか行きたいです。
