Pythonで区切り文字の最後の要素を取得

February 16, 2022

Pythonで、区切り文字の最後の要素を取得する。「要素」と言ってるのは、一旦リストにする必要があるため。

 

例えば、'/path/to/document' から、“document"だけ抽出したい場合。

 

#文字列を一旦リストにする
str = '/path/to/document'
doc_list = str.split('/')
print(doc_list)
['', 'path', 'to', 'document']

#[-1]により、リストから最後の値を取得
doc_name = doc_list[-1]
print(doc_name)
document
