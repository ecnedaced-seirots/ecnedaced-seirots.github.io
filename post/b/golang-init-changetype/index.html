Go入門(11) - 型変換(str-int, int-str)

February 2, 2022

文字列から数値 or 数値から文字列。Goでこのように型を変更するには、strconvを使う。

 

changetype.go

package main

import (
    "strconv"
    "fmt"
)

func main() {
    //文字列から数値に変換(string -> int)
    var i int
    var str string="100100"
    i, _ = strconv.Atoi(str)
    
    //変換後の値を出力
    fmt.Println(i)
    
    //変換後の型を出力
    fmt.Printf("%T \n", i)

    fmt.Println("---------------")

    //数値から文字列に変換(int -> string)
    var i2 int=200200
    var str2 string
    str2 = strconv.Itoa(i2)
    
    //変換後の値を出力
    fmt.Println(str2)

    //変換後の型を出力
    fmt.Printf("%T \n", str)
}

 

結果

100100
int 
---------------
200200
string

 

ちなみに変数の定義方法は以下でもいいのだが（ただし関数内に限る）、どっちにしても3行必要なんだな。

    i   := 100100
    var str string
    str = strconv.Itoa(i)
